1)	PULL THE IMAGES: sudo docker pull mysql:(tag)
2)	CHECKING/LISTING IMAGES: sudo docker images
3)	CHECKING/LISTING CONTAINERS: sudo docker ps
4)	CHECKING /LISTING ALL CONTAINERS (INCLUDING EXITED): sudo docker ps -a
5)	RUNNING THE CONTAINERS WITH IMAGE: sudo docker run -it (container_name) /bin/bash
6)	MODIFYING THE CONTAINERS: sudo docker exec -it (container_name) /bin/bash
7)	GETTING THE IP ADDRESS: sudo docker inspect
8)	TRANSFER THE FILES FROM HOST TO CONTAINER: 
	sudo docker cp (source_file) container:(dest_file)
	sudo docker cp container:(dest_file) (source_file)
9)	CREATE CONTAINERS WITH IMAGE: sudo docker create mysql:(tag)
Actual Command  sudo docker run -d --name=test-mysql -e MYSQL_ROOT_PASSWORD=’mypassword’ -v /var/lib/mysql mysql:5.6
10)	SCP FROM LOCAL LAPTOP TO COE MACHINE USING WINSCP
11)	COPYING THE FOLDER FROM LOCAL HOST (COE MACHINE) TO SERVER: scp -r tpcc-mysql-master dockuser@129.10.58.161:/home/dockuser/dockpro
12)	COPYING THE FILES FROM HOST SERVER TO CONTAINER: sudo docker cp /home/dockuser/dockpro/tpcc-mysql-master proj-mysql:/home
13)	INSTALL VI EDITOR IN ALL CONTAINERS: 
	sudo docker exec -it (container_name) /bin/bash
	apt-get update
	apt-get install vim
